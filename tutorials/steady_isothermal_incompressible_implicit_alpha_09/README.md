This folder contains validated GUI tutorial scenes for the current
steady, incompressible, isothermal solver using implicit pressure
relaxation with `alpha = 0.90`.

Validated cases:

- `01_pipe_only`
- `02_fittings_no_elevation`
- `03_fittings_with_elevation`
- `04_inlet_flow`
- `05_outlet_flow`
- `06_inlet_and_outlet_flow`

These scenes use:

- implicit pressure relaxation
- `alpha = 0.90`
- `velocity_loop_method = secant`
